question: Нужно ли настраивать каналы для личных сообщений?
---

Для обычного Direct Message отдельный #канал не всегда нужен: личное сообщение можно отправить выбранной ноде из приложения.

Но для приватности лучше использовать отдельный зашифрованный канал:
- Создать Secondary Channel
- Сгенерировать уникальный #ключ
- Передать QR-код только нужным участникам
- Не публиковать ключ в открытом чате

Стандартный общий ключ подходит для совместимости, но не для приватного общения.

Если ключи не совпадают, ноды могут видеть друг друга, но сообщения не будут нормально читаться.
