question: Как создать отдельный зашифрованный канал?
---

Создайте Secondary Channel с уникальным #ключом.

Общий порядок:
1. Откройте настройки каналов
2. Добавьте новый канал
3. Задайте имя канала
4. Сгенерируйте новый PSK
5. Поделитесь QR-кодом только с нужными участниками

Не публикуйте QR-код приватного канала в открытом чате. Все, у кого есть ключ, смогут читать сообщения этого канала.

Для общей сети отдельный приватный канал не заменяет базовые LoRa-настройки: #регион, пресет и частотный слот всё равно должны совпадать.
