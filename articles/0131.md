question: Какие настройки MQTT использовать для Калининградской области?
---

Базово:
- MQTT включен: `Да`
- Шифрование включено: `Да`
- JSON включен: `Нет`
- TLS включен: `Да`
- Client Proxy: `Да`, если нода через #bluetooth
- Map Reporting: `Да`
- LoRa OK to MQTT: `Да`
- Uplink на канале: `Да`
- Downlink: по необходимости
