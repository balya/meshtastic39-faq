question: Как использовать Range Test для проверки дальности?
---

Range Test нужен для временной проверки дальности между нодами.

Правила:
- Используйте его только на время теста
- Не оставляйте включённым в общем канале
- Лучше тестировать двумя нодами на фиксированных точках
- Записывайте место, высоту, #антенну, RSSI/SNR и результат
- По возможности отключайте #mqtt, если проверяете радио

В общем канале Range Test засоряет эфир и мешает обычной работе сети. Для регулярной связи он не нужен.
