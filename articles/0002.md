question: Работает ли Meshtastic без интернета, сотовой связи и SIM-карты?
---

Да. Meshtastic может работать полностью без интернета, сотовой связи, SIM-карты и #mqtt.

Сообщения передаются по LoRa-радиоканалу между нодами. Телефон нужен только как экран, клавиатура и приложение для настройки, если у устройства нет собственного интерфейса.

Интернет может использоваться дополнительно:
- Для #mqtt
- Для карт
- Для связи удалённых сегментов сети
- Для ботов и внешних сервисов

Но для обычной радиосвязи между нодами интернет не обязателен.
