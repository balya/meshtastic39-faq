question: Как работает приложение Meshtastic на Android?
---

Приложение на #android подключается к ноде по #bluetooth, USB или сети и служит интерфейсом управления.

Через него можно:
- Читать и отправлять сообщения
- Настраивать ноду
- Смотреть список нод
- Работать с картой
- Передавать координаты телефона
- Использовать Client Proxy для #mqtt

Для стабильной работы выдайте нужные разрешения, отключите жёсткое энергосбережение для приложения и не держите одновременно несколько клиентов, подключённых к одной ноде.
