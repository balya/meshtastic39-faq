question: Какие базовые настройки нужны для начала работы в местной сети?
---

Для первой настройки используйте значения из закреплённых настроек сообщества.

Базовый набор:
- Регион: `RU`
- Шаблон: `LONG_FAST`
- Количество прыжков: `7`
- Частотный слот: `2`
- Частота: `869.075`, только если нужно вручную исправить сбившееся значение

Для общего канала должны совпадать #частота, пресет, слот и #ключ канала. Если хотя бы один из этих параметров отличается, ноды могут не услышать друг друга или не смогут прочитать сообщения.

#mqtt настраивается отдельно и не нужен для первой проверки LoRa-связи. Сначала лучше проверить связь между двумя нодами по радио, а уже потом подключать карты, бота и интернет-шлюз.
