id: 0256
question: Как сменить регион Apple ID для установки приложения Meshtastic?
---

Если приложение Meshtastic недоступно в вашем регионе App Store, можно сменить регион Apple ID или использовать отдельный Apple ID.

Перед сменой региона проверьте:
- Активные подписки
- Баланс Apple ID
- Способ оплаты
- Семейный доступ
- Доступность других нужных приложений

Часто проще создать отдельный Apple ID для установки приложения из другого региона.

После установки приложение можно использовать с нодой по #bluetooth. Интерфейс и доступность в App Store могут меняться.
