question: Нужно ли отключать шифрование для работы MQTT?
---

Нет. Для обычной работы #mqtt шифрование отключать не нужно.

В Meshtastic сообщения канала остаются зашифрованными, если используется PSK. MQTT просто переносит данные через интернет.

Отключать #шифрование опасно:
- Сообщения становятся читаемыми посторонним
- Можно нарушить приватность канала
- Licensed/HAM режим меняет правила работы сети

Для публичных интеграций иногда используют JSON, но такие данные не шифруются. Включайте JSON только если понимаете последствия.

Обычная рекомендация: MQTT включить, шифрование оставить включённым.
